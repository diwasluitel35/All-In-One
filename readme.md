# All In One

This repository contains combined demos of all NLP models I trained.


## Model
All trained models are uploaded to Hugging Face:

[Hugging Face Models](https://huggingface.co/diwasluitel/models)

## Demo Usage
To run the demo:

1. Download this repository.
2. Open a terminal in the repository folder.
3. Run the Flask server:
```
python master.py
```
4. After it runs, open your browser and goto: http://127.0.0.1:5000

## UI Development

The front-end design was created with assistance from code generation tools to speed up development, allowing the core project focus to remain exclusively on the back-end NLP model and evaluation process.

## Installation

Before running the demo, ensure you have necessary dependencies installed. All dependencies are listed in **requirements.txt** file. Install them with:
```
pip install -r requirements.txt
```
